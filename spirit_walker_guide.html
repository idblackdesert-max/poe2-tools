Patch 0.5 Rule ใหม่: Crafting methods ที่ add guaranteed modifier เดียว จะ add มันในรูปแบบ "Crafted Modifier" (สีน้ำเงินในเกม) — items มีได้แค่ 1 Crafted Modifier เท่านั้น ต่อ item

กฎหลักที่ต้องจำ

  • 🔵 Crafted Modifier = modifier สีน้ำเงิน — มาจาก Essence หรือ Alloy
  • ⚠️ Item มี Crafted Modifier ได้แค่ 1 ชิ้นเท่านั้น
  • ✅ ถ้าใช้ Essence บน item แล้ว → item นั้น ใช้ Alloy ซ้ำไม่ได้อีก
  • ✅ วาง Crafted Modifier บน slot ที่ยังขาด mod สำคัญที่สุด
  • 📌 Crafted Modifier ถูก lock — ไม่หลุดจาก Orb of Chaos / Annul ปกติ

ตัวอย่างการใช้ถูก

→ ไม่ต้อง Alloy บน Body Armour อีก (slot crafted เต็มแล้ว)

2H Mace: ใส่ Alloy ที่ให้ Physical Damage
→ ไม่ต้อง Essence บน Mace แล้ว

ตัวอย่างการใช้ผิด

❌ ใช้ Essence บน item → แล้วพยายาม Alloy บน item เดียวกัน
→ ทำไม่ได้ — Crafted Modifier slot เต็มแล้ว

Alloy System — Act 2+ (สำคัญมาก)

Alloys 13 ชนิด unlock หลัง Farrow's Quest Act 2 — เก็บ Remnant encounters ไปด้วยระหว่างเดิน

วิธีใช้ Alloy

  1. ต้องเป็น Rare item (สีเหลือง) เท่านั้น — Magic item ใช้ไม่ได้
  2. Alloy ลบ 1 mod แบบสุ่ม ออกก่อน แล้วใส่ mod ใหม่ที่การันตีเข้ามา
  3. ผลลัพธ์คือ Crafted Modifier สีน้ำเงิน — slot crafted เต็มแล้ว
  4. ใช้บน item ที่ mod ดีๆ ครบแล้ว — อย่าเสียถ้า item ยังต้องการ mod อื่น

Alloy ที่น่าใช้สำหรับ build นี้

Alloy ที่ให้ Minion/Companion Damage บน weapon
Alloy ที่ให้ +Spirit บน Body Armour

Imbued Alloy (elemental ailment) — ไม่มีประโยชน์กับ build
Swift Alloy (flask charges) — ใช้ได้บน Belt ถ้าอยากได้
Runic Alloy — ใส่บน Ring/Amulet สำหรับ Runic Ward
⚠️ Alloy 13 ชนิดยังไม่เปิดเผยทั้งหมด — เช็ค tooltip ในเกมก่อนใช้เสมอ

  • Fire Resist 75%
  • Lightning Resist 75%
  • Cold Resist 75%
